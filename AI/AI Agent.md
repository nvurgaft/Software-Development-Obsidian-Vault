An AI agent is a system that can perceive its environment, make decisions, and take actions to achieve specific goals. Unlike a traditional program that simply executes predefined instructions, an AI agent can analyze information, reason about it, and choose actions dynamically using an LLM model.

The “environment” could be many things: a computer system, the internet, a database, a game world, or even a physical robot interacting with the real world.

An LLM provides language understanding and reasoning, while an AI agent provides decision-making, planning, memory, and the ability to take actions.
