A data warehouse is a central repository or data belonging to an organization, it is a core component of that organization's business intelligence. It can accept data from multiple sources (departments, branches, etc..) and organized it for easy retrieval, analysis and reporting.

A data warehouse helps businesses analyze historical data, find emergent trends and performance issues, it helps with generating data for reports and decision making.

Data warehousing is the process  of collecting, storing and managing data from different sources into the central repository. The main workflows for building a Data warehouse are [[ETL (Extract, Transform, Load)]] and [[ELT (Extract, Load, Transform)]].
