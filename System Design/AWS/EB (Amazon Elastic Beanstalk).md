Elastic Beanstalk is an orchestration services used for deploying applications which orchestrate various other AWS services, including [[EC2 (Elastic Compute Cloud)]], [[S3 (Simple Storage Service)]], CloudWatch, Elastic Load Balancers.

Elastic Beanstalk provides an additional layer of abstraction above manually configuring, deploying and scaling clusters of web applications.
Developers upload their their application code and Elastic Beanstalk automates the provisioning of resources, load balancing, scaling and the monitoring of these applications.
