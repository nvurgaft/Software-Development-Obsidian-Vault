ECS is a managed service that enables developer to fully run and orchestrate containerized applications across a cluster of [[EC2 (Elastic Compute Cloud)]] instances.
It simplifies allocation, deployment and scaling of clusters on containerized applications.

It runs instances in isolated VPC environments, support IAM role based access, secrets manager (securely store passwords and keys) and data encryption.

* Configurations include, choosing the Docker image, 
* CPU allocation
* Ports
* Environment variables, logging, IAM roles

Common use cases are 
* Microservices Architecture.
* Provisioning resources for batch processing and their execution.
* Machine learning by deploying ML models into containers.
